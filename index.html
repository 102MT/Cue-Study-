큐카드 (과목 선택 + Anki 스타일)

- 과목(덱)을 선택해서 공부합니다.
- 카드 입력 형식: 앞면 || 뒷면 (한 줄당 1장). 줄바꿈은 \\n 사용 가능.
- 단축키: Space 뒤집기 · ←/→ 이동 · A/H/G/E 평정(Again/Hard/Good/Easy)
GitHub Pages를 쓴다면 Public이라 기본적으로 공개 웹페이지입니다. 개인정보/민감정보는 카드에 넣지 마세요.

앞면
평정은 “우선순위 점수”를 조정합니다. Again일수록 더 자주 나오게 됩니다(간단 Anki식).
※ 진짜 Anki(SM-2) 수준의 “일자 기반 스케줄링”도 구현 가능하지만, 여기서는 가볍게 세션 중심으로 구성했습니다.
아래에 앞면 || 뒷면 형식으로 붙여넣고 “적용”을 누르세요. 뒷면에 줄바꿈이 필요하면 \\n을 사용하세요.
